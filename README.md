# SauceDemo Automation Testing

## 📌 Overview

This project demonstrates basic automation testing of the SauceDemo web application using Selenium WebDriver and TestNG.

It was created as part of a QA practical assignment to validate core user flows and identify functional issues.

## 🔗 Application Under Test

https://www.saucedemo.com/

## 🧰 Tech Stack

* Java
* Selenium WebDriver
* TestNG
* Maven

## ✅ Test Scenarios Covered

* Valid Login
* Invalid Login
* Product Listing Validation
* Add Product to Cart
* Checkout Flow

## ▶️ How to Run

1. Clone the repository
2. Open in IntelliJ or Eclipse
3. Run the test class:
   mvn test

## 📊 Observations

* Checkout allows submission without proper field validation
* Error messages can be improved for better clarity
* Minor delay observed when adding items to cart
